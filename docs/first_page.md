## 18. Functions

#### 1. Draw in a single Geogebra notebook the following functions: 
- $f(x) = x^2$
- $g(x) = x$
- $h(x) = \frac{1}{x}$
- $j(x) = \sin(x)$

Find value of all the above functions at $x = 2$.

- $f(2) = 2^2 = 4$
- $g(2) = 2$
- $h(2) = \frac{1}{2} = 0.5$
- $j(2) = \sin(2) \approx 0.909$

Final Values at $ x = 2$:
- $f(2) = 4$
- $g(2) = 2$
- $h(2) = 0.5$
- $j(2) \approx 0.909$
